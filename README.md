# TwitterSched - Front

## Getting started
```
yarn build --production
```
Delivers compiled files in dist/*

## Docker image
Build docker image
```
docker build -t germainlefebvre4/twittersched-front .
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
